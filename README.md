<details>
<summary>¿Cómo matricular créditos adicionales?</summary>

  Una vez matriculado el periodo académico o generada y pagada la primera factura del periodo vigente bajo el beneficio del Ministerio   de Educación Nacional, se podrá realizar una segunda factura si el estudiante desea matricular más; podrá hacerlo desde la opción     "Créditos adicionales" bajo recursos propios.

Todo estudiante puede matricular hasta 21 créditos, por lo que, para esta opción se activará la cantidad excedente, de acuerdo con lo matriculado bajo el convenio. (Ejemplo: si matricula 14 créditos con beneficio, el sistema activará hasta 7 créditos para completar la cantidad máxima por periodo).

Esta opción se sugiere realizarla después de hacer el pago de la primera factura (la factura con Política de Gratuidad).
</details>
